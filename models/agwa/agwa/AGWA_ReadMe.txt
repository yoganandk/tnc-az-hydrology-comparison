This is the AGWA folder required for using the AGWA Urban Addin in ArcMap.
Contents:
1) ADDINS - Includes the latest version of the AGWA Urban addin. Double click to install add in for ArcMap v10.8
2) Data - Includes input data used to create the model.
3) datafiles - Files required for the correct functioning of the AGWA tool
4) documentation - AGWA Tool documentation
5) logginh - Empty folder to be used for logging AGWA messages
6) models - All model executables that can be run using AGWA
7) workspace - 
	- SierraVista - folder that stores outputs from the AGWA Urban tool. Includes KINEROS2 text files.
	- SierraVista.mdb - Geodatabase created by the AGWA Urban tool
	- LaTerraza.mxd - ArcMap Map Document used to create and execute the KINEROS2 model.
